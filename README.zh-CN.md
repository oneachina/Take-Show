# TakeShow 🧪✨

[English](README.md)

TakeShow 是一个轻量、偏原版向的扩展模组：专注把 Minecraft 中“延伸玩法较少、存在感偏弱”的内容做成更实用、更有互动的玩法补全，而不是引入庞大复杂的新系统或用大量内容堆满世界。

本项目目前仍处于 **不完整版 / 开发中** 🚧，后续可能会继续扩展并进行平衡调整。

## 功能（当前已添加）✨

- **回响护符（Echo Charm）** 🧿：右键触发一次“回声 Ping”，让附近生物短暂发光，并提供粒子/音效反馈与冷却。
- **回响紊乱（Echo Distortion）** 🌀：与回声能力绑定的代价效果（移动速度降低）。
- **鸣响镐（Resonant Pickaxe）** ⛏️：挖掘石质方块时扫描一定范围内的“隐藏矿石”（未暴露在空气中的矿），并进行短暂高亮显示。
- **附魔：瞅什么瞅？！（Look What?!）** 🛡️：最高 III 级的防御/反制向附魔，被攻击时施加减益；高等级在举盾时提供更强反制能力。
- **联机友好** 🌐：由服务端筛选目标与坐标下发，客户端负责短时高亮渲染。
- **进度（Advancements）** 🎯：少量进度用于引导首次使用。

## 依赖 🧵

- Fabric Loader
- Fabric API
