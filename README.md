# TakeShow 🧪✨

[简体中文](README.zh-CN.md)

TakeShow is a lightweight, vanilla-friendly expansion mod focused on one idea: give “underdeveloped” Minecraft items meaningful new utility and interactions—without introducing huge systems or content bloat.

This project is still **incomplete / in development** 🚧, so features and balance may change.

## Features ✨

- **Echo Charm** 🧿: right-click to trigger an “echo ping” that briefly highlights nearby living entities (glowing), with particles/sound feedback and a cooldown.
- **Echo Distortion** 🌀: a harmful status effect used as the trade-off for echo-related utility (reduced movement speed).
- **Resonant Pickaxe** ⛏️: when breaking stone-like blocks, pings nearby *hidden* ores (ores not exposed to air) and highlights them for a short time.
- **Look What?! Enchantment** 🛡️: defensive enchantment (up to level III) that debuffs attackers; at higher levels adds stronger counterplay while blocking with a shield.
- **Multiplayer-friendly networking** 🌐: server finds targets/positions and sends them to the client for short-lived highlight rendering.
- **Advancements** 🎯: a small set of advancements to guide first-time use.

## Dependencies 🧵

- Fabric Loader
- Fabric API

## Multiplayer 🌐

For the best experience, install on **both client and server**.
